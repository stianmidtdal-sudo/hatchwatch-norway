How it works
Dual Trigger Model

Both triggers run in parallel, each blending real observations with a forward projection. GDD accumulates from actual daily temperatures, then projects using 10-year historical averages to the species threshold. Water temperature is extrapolated from a live trend to the 12°C hatch threshold. The estimated hatch date is where the last of the two conditions is met.

GDD · Growing Degree Days

Heat accumulates from January 1st. Each species crosses a calibrated thermal threshold — that crossing is the forecast.

Stefan Ice Model

Ice-out is predicted using Stefan's law — a physical growth/melt model calibrated against satellite-observed records from each location.

Data Sources

Air temperature and snow cover from MET Norway's Frost API. Water temperature from NVE HydAPI. Updated daily.
